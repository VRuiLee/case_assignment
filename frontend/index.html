Password Requirements

  • The input must be between 8 and 20 characters long.
  • It must contain at least one uppercase letter, one lowercase letter, one digit, and one special character (e.g., @, #, $).
  • It should not contain any whitespace.
  • It must not match any of the following banned words: password, admin, 12345678, letmein, welcome.
  • Duplicate inputs should not be allowed.
